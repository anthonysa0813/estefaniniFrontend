# Frontend miSaludApp

- La farmacéutica “Mi Salud” desea llevar el control de su inventario, para lo cual,recurre a sus servicios para que realice la implementación. Como punto de partida necesita que lleve a cabo una prueba de concepto para asegurar que el requerimiento es viable y tenga éxito. El requerimiento es el siguiente:

- Para correr el proyecto, es necesario realizar

```bash
  npm install
```

Esto se debe a que se necesita instalar las dependencias que utiliza este proyecto, Por ejemplo:

- react-router-rom
- react-icons
- styled-components

- Por último, para correr la aplicación.

```bash
npm run dev
```

- Recomiendo correr primero el proyecto del backend: https://github.com/anthonysa0813/backendEstefanini
